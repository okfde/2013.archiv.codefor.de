Code a difference: Nutz' Deine Fähigkeiten um Deine Stadt zu verbessern!
{% include h4c-map-legend.html %}

Jede Woche treffen sich in ganz Deutschland Teams zu ihren Projekten für Code for Germany. Sie arbeiten gemeinsam an nützlichen Anwendungen und Visualisierungen rund um offene Daten und digitale Werkzeuge für Bürgerinnen. Diese lokalen Gruppen und ihr Treffen nennen wir Labs.

Die Labs bestehen aus Interessierten zu Open Data. Dazu gehören auch Programmiererinnen und Designern. Sie erproben, wie Daten und Software dabei helfen können die Städte und Nachbarschaften zu verbessern. Dazu wird Code für Dienste geschrieben und bestehende Daten so aufbereitet, dass sie für alle genutzt werden können.

Wo gerade welche Anwendungen bei welchen Projekten entstehen, wann das nächste Treffen stattfindet und wie Du dort mitmachen kannst, erfährst Du nach einen Klick auf das Lab der jeweilige Stadt.
