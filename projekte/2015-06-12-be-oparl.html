--- layout: project lab: - OK Lab Berlin - OK Lab München - OK Lab Ruhrgebiet title: OParl imgname: berlin/oparl.png status: Released archive: false category: politics links: - url: https://oparl.org name: Projektwebseite - url: https://github.com/OParl/ name: GitHub - url: https://dev.oparl.org/ name: Entwicklerplattform collaborators: - name: Stefan Graupner (Berlin) - name: Ernesto Ruge (Ruhrgebiet) - name: Konstantin Schütze (München) --- OParl ist eine Initiative zur Standardisierung des offenen Zugriffs auf parlamentarische Informationssysteme in Deutschland. Das Ziel von OParl ist die Schaffung einer Standard-API für den Zugang zu öffentlichen Inhalten in kommunalen Ratsinformationssystemen, damit die Inhalte daraus im Sinne von Open Data für möglichst viele verschiedene Zwecke eingesetzt werden können.
