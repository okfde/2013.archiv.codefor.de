--- layout: project lab: OK Lab Berlin title: OParl links: - url: http://oparl.org name: Projektwebseite - url: http://github.com/OParl/ name: GitHub - url: http://demoserver.oparl.org name: Demo colaborators: - name: Lucas Jacob - name: Stefan Graupner - name: Tim Scheuermann --- OParl ist eine Initiative zur Standardisierung des offenen Zugriffs auf parlamentarische Informationssysteme in Deutschland. Das Ziel von OParl ist die Schaffung einer Standard-API für den Zugang zu öffentlichen Inhalten in kommunalen Ratsinformationssystemen, damit die Inhalte daraus im Sinne von Open Data für möglichst viele verschiedene Zwecke eingesetzt werden können.
