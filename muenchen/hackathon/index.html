Hackathon zum Open Data Day

Am 21.Februar 2015 ist Open Data Day - weltweit. In München veranstalten Code for München und die Stadt München einen Hackathon. Wir laden Euch ein, an einem von drei Projekten mitzuarbeiten:

  • Finder-App (wo finde ich einen Spielplatz, Bankautomat etc.)
  • München Transparent: wie funktioniert Stadtpolitik?
  • Meta-Suchmaschine für Wohnungen in München
Wir wollen Hackathons auch für Fortbildung nutzen: Deswegen gibt es eine Einführung in die Datenvisualisierung, die sich vor allem an Journalisten richtet.

Unser Gastgeber, die Landeshauptstadt München, startet beim Hackathon offiziell ihr Open-Data-Portal.

Ihr habt Lust, dabei zu sein? Dann meldet Euch kostenlos auf unser MeetUp-Seite an. Gerne könnt Ihr mit diesem Flyer Werbung für den Hackathon machen. Zum ersten Code for München-Hackathen am 29. November 2014 gibt es einen Rückblick auf Storify.
